DesktopNoteSaver:
 Written primarily in Python, thid program is bound to a hotkey and can be
accessed on-demand. Pressing of the hotkey opens a text document containing all text that is
currently on the users desktop image. The text can hen be edited and stored. Upon saving the
notepad document, your desktop background image will be updated with the latest text.

This program allows the user to take notes without the need for a program to run
24/7 (example: windows “sticky
notes”program).

TwitterArchiver:
This was a Python script that I wrote to archive tweets posted by specifix users.
The purpose of this program is to potentially save
large amounts of data to later be evaluated. This project was the start of my research
into Big Data.

Charity-League:
This was a responsive web app that I designed at the 2016 UPS hackathon. The team and I
stayed up all night to develop this application within 28 hours. We used meteor and
meteor kitchen to build this application from the ground up. These technologies made use of mongoDB to manage our database requiring information.
Our web application first prompts the user with 3 forms of verification. Charity, Company,
and Employee. The charity login lets you register a charity and post events that your charity is
hosting. These events are able to be toggled on and displayed to the companies. Companies can then
see these events and choose which ones to make available to their employees. Once the company toggles
an event on, the employees can now see events. They can gain points by doing different event related
activities. We also launched a mobile android app to display this information in a mobile app design
for those users that did not like viewing it within a browser. We placed second place in the Hackathon.

ClientServerGame:
 This project was a Client/Server game with multi-threading, mutex protection, user
authentication, account creation, ranked scoreboard, random matchmaking, invitation
matchmaking, ranked and unranked games (games that counted towards high scores). The
project was soon to also include a chat feature to allow users to communicate, however,
my focus eventually shifted."
